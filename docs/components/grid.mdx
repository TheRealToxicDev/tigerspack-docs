## API

|Prop          |Type      |Available options                    |Description                                 |Default       |
|--------------|----------|-------------------------------------|--------------------------------------------|--------------|
|**align**     |string    |```left``` ```right``` ```center```  |Horizontal position of child elements       |``` left ```  |
|**children*** |jsx       |                                     |The content                                 |              |
|**flow**      |string    |```row``` ```column```               |Flow of child elements                      |```row```     |
|**padding**   |number    |```0 - 1000```                       |The indent within the block                 |``` #000 ```  |
|**stretch**   |string    |```width``` ```height``` ```full```  |Stretches the width by 100%                 |``` #fff ```  |
|**size**      |number    |```0 - 12```                         |Cell size                                   |```auto```    |
|**valign**    |string    |```top``` ```bottom``` ```center```  |Vertical position of child elements         |```top```     |
|**wrap**      |bool      |```true``` ```false```               |Allows you to move elements to the next line|```false```   |
